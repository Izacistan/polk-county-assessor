    Parcel Helpclose
    The easiest thing to do is to cut and paste a district/parcel or a geoparcel into this field.The parcel number can be formatted with dashes and slashes, but does not have to be. It can also be abbreviated. Here are some examples of district/parcel entries that will work:
    • 090/06110-001-000
    • 09006110001000
    • 090.6110.1
    • 090/6110-1
    • 171/00511-331-001
    • 171/511.331.1
    If you don't want to select a single parcel, but want to match a set of parcels, use a formatted string and end with a dot. A dot at the end doesn't add leading zeros. This method only works for the first and second parts after the district.
    • 1st part: Starts with 090/061 - Use 090.061.
    • 2nd part: Starts with 241/523-067 - Use 241/523-067.
    You can also indicate that you want to select a range of parcels.
    • 241/523-67-200:241/523-67-299
    • 090/6100..090/6200
    All of the above also apply when using a geoparcel instead of a district/parcel.
    Street Address Helpclose
    • Enter the street address into this field. Do not include the city, state, or zipcode.
    • You can cut and paste an address into the box.
    • You can enter the full street address (1431 E 32nd St) or any part of the street address.
      • If you enter just a single word (or part of a word), it will use that as the beginning of the street name.
      • However, if that word happens to be a street direction, it will also find all properties with that direction.
      • If you enter only a number, it will be a house number not a street name.
      • If you want all properties on 1st, you must enter 1st.
      • If you include a house number and a street name, you can use just the number for the street.
      • You can type a unit at the end of an address by using the #, Suite, Ste, Unit, Apartment or Apt.
    • Address Ranges
      • Use a : (colon) or .. (two dots) between two house numbers to specify a range.
      • For example, 3000..4000 River Oaks will return all addresses between 3000 and 4000 River Oaks.
    • Wildcards in the Street Name
      • % (percent) matches any number of characters.
      • _ (underscore) matches a single character.
      • Martin%King% will find all streets starting with Martin containing King.
      • Gar _ _ _ will find all streets that start with Gar followed by three characters.
    • The link, Street Names, will help you find the street names used in Polk County.
